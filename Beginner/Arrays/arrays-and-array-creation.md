Q) What is an array and how do you create an array?

Ans)
    An array is a data structure used to store multiple values in a single variable.
    The values in an array are stored in an ordered list and can be accessed using an index.

    💻 Code Examples
        🔹 Creating an Array (Array Literal)
                let fruits = ["apple", "banana", "mango"];

                console.log(fruits);

        🔹 Creating an Array Using new Keyword
                let numbers = new Array(1, 2, 3, 4);

                console.log(numbers);

        🔹 Accessing Array Elements
                let colors = ["red", "blue", "green"];

                console.log(colors[0]); // red
                console.log(colors[1]); // blue

    🔍 Explanation 

            Arrays store multiple values in one variable
            Elements are accessed using index numbers
            Index starts from 0

    🎯 One-Line Interview Answer

         An array is a data structure that stores multiple values in a single variable using indexed positions.
